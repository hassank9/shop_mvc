A modern full-featured e-commerce website built with Native PHP, MySQL, AJAX, and Bootstrap, including a professional admin dashboard for managing products, categories, brands, orders, settings, and hero sections.

## Arabic

### نبذة عن المشروع
**Kallashy & Kallashy** هو مشروع متجر إلكتروني متكامل تم تطويره لتقديم تجربة تسوق عصرية، أنيقة، وسهلة الاستخدام.  
يتيح الموقع للزائر تصفح المنتجات، البحث والتصفية، إضافة المنتجات إلى السلة، إتمام الطلب، ومتابعة تجربة شراء واضحة وسلسة من البداية إلى النهاية.

## English

### Project Overview
**Kallashy & Kallashy** is a full-featured e-commerce website built to deliver a modern, elegant, and user-friendly shopping experience.  
The platform allows users to browse products, search and filter items, add products to the cart, place orders, and enjoy a smooth purchasing journey from start to finish.

The project is not limited to the customer-facing store only — it also includes a **professional admin dashboard** for managing products, categories, brands, orders, settings, hero slider content, and key website sections. This makes it a practical and scalable solution that can grow into a real commercial product.

---

### Project Vision
This project was built to create a modern e-commerce system that combines:
- Attractive design
- Smooth user experience
- Performance and speed
- Flexible management
- Future scalability

The goal was not only to build a visually appealing storefront, but also to develop a clean and organized code structure that is maintainable and easy to extend.
